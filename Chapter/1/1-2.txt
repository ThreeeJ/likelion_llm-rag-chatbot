# 강의 제목 : Chat GPT와 LLM에 대한 기초 이해

## Keyword
- 머신러닝, 딥러닝, LLM, 생성형 AI, Attention, 추론, 프롬프트 엔지니어링

### 모델 : 소프트웨어 프로그래밍 언어로 작성된 알고리즘

---

## 머신러닝 (Machine Learning)
- 컴퓨터가 무언가에 대한 의사결정(예측)을 위해 데이터를 기반으로 학습하는 것
- "기계가 데이터를 학습시키도록 하는 것"
- 나오게 된 배경: 과거의 데이터를 기반으로 미래를 얼마나 잘 예측하는가에 대한 필요성 증대
- 챗GPT의 원리: 데이터를 기반으로 기계가 다음을 예측하는 것
- 컴퓨터가 사물을 구분하는 방법
  - 직접 알려줘야 한다, 이게 무엇이다 하고 알려줘야 한다
  - "데이터 라벨링" 이라고 부른다

### 종류
  - 지도 학습
    - 문제와 정답을 모두 알려주며 학습시키는 방법
  - 비지도 학습
    - 정답 없이 데이터의 패턴을 스스로 학습하는 방법
  - 강화 학습
    - 보상을 통해 상은 최대화하고, 벌은 최소화하는 방향으로 행동을 강화하며 학습하는 방법

---

## 딥러닝 (Deep Learning)
- 머신러닝의 한 종류로, 인공신경망을 기반으로 함
- 비정형 데이터로부터 특징 추출 및 판단까지 기계가 한 번에 수행 가능
- 사람이 사고하는 방식처럼, 여러 층을 통해 단계별로 예측하도록 기계를 학습시키는 모델

---

## LLM (Large Language Model)
- AI 모델? 학습된 기계, 즉 머신러닝 소프트웨어가 탑재된 기계
- Language Model? 사람의 언어를 인식하고, 그 언어를 기반으로 답변을 생성할 수 있는 모델

---

## 중간요약
1. 머신러닝은 데이터를 주고 기계를 가르치는 작업
2. 기계를 가르치는 목적은 "예측"
3. 학습 과정에는 데이터가 필수
4. AI Model을 만들기 위해서는 데이터가 필요
