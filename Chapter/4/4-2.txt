# 강의 제목 : React를 사용한 UI 개발

---

## 1. 함수형 컴포넌트
- 함수형 컴포넌트는 자바스크립트 함수로 작성하는 리액트 컴포넌트

## 2. 상태 관리 - React hook
- 아주 중요!!, 프론트엔드에 핵심
- 모든 컴포넌트는 상태를 가지고 있다
  - 상태(State): 컴포넌트가 기억해야 할 동적인 데이터(예: 사용자 입력, 서버 데이터, UI 상태 등)를 의미
- hook : 자동으로 상태를 변환하게 도와주는 툴
  - 클래스를 사용하지 않는 함수형 컴포넌트에서 상태(state)와 생명주기(lifecycle) 기능을 사용할 수 있게 해주는 특별한 함수

## 3. React Props
- 부모 컴포넌트에서 자식 컴포넌트로 데이터를 전달할 때 사용하는 읽기 전용 데이터
