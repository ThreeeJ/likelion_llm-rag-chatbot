# 강의 제목 : 사용기술 설정

---

## 해당 기능을 구현하기 위한 기술 스택 선택
-> 책임감을 가지고 조사를 해야한다!

ex) 개발자를 위한 기술 면접 준비 챗봇 개발 - 핵심 기능 및 부가 기능의 분류

핵심 기능
- 유저는 개발 관련 질문을 하고 원하는 답변을 받을 수 있다 -> OpenAI API, Anthropic API...

부가기능
- 대화 맥락 지정 -> Redis
- 데이터베이스 기반 자료 검색 -> RAG
- 채팅 기록 및 데이터 저장 -> Postgresql, MySQL
- 채팅 검색 -> Elastisearch
- 회원가입 -> Google Auth
- LLM 모델 변경 -> Unleash
- 사진 업로드 -> AWS S3
