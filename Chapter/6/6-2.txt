# 강의 제목 : RAG 아키텍처 이해

## 명령어(윈도우)
접속 : docker exec -it redis-stack redis-cli

---

## 검색증강생성을 위해 필요한 사항들
1. 검색 출처 파일(txt, PDF 등)
2. 검색 텍스트 벡터 임베딩
3. 유저의 질문 벡터 임베딩
4. 벡터DB에서 유사도 검색

---

## Redis vector DB 관련 로직 추가
1. 벡터 유사도 검색 최적화를 위한 인덱스 생성
2. 문서의 텍스트를 벡터 임베딩하여 Redis에 저장
3. 사용자 질문과 유사한 벡터 데이터 찾기

## RAG를 통한 답변 생성 로직
1. OpenAI API로 사용자 질문을 벡터 임베딩
2. 1번 결과를 통해 vector DB 유사도 계산
3. 유사도 높은 문장 추출
4. 3번 결과를 통해 Prompt Context Providing
